## 新增文章的完整流程

### 一般文章
1. 建立 `blog/YYYYMMDD+類型+編號/index.html`
2. 更新 `blog/index.html` 卡牆（最上面加卡片）
3. 更新 `sitemap.xml`（加一筆 `<url>`，首頁和 blog/index 的 `<lastmod>` 也更新）
4. 更新 `README.md`（對應分類加一行）

### Q&A 文章（多一步）
1. 建立 `blog/YYYYMMDD+qa+編號/index.html`
2. 更新 `blog/index.html` 卡牆（最上面加卡片）
3. 更新 `qa/index.html` 列表（最上面加卡片）
4. 更新 `sitemap.xml`
5. 更新 `README.md`（Q&A 分類加一行）
