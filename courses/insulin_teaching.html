第一頁 · 基本概念

胰島素，
身體的「合成型荷爾蒙」

不只是碳水化合物會觸發——進食就會分泌，而且它在線上時，燃脂反應會被暫停。
  • 1
    進食觸發分泌
    碳水最強、蛋白質中等、脂肪與碳水同吃時會放大反應。甚至聞到食物氣味，大腦就會預先觸發少量分泌。
  • 2
    它的工作：開門讓葡萄糖進細胞
    胰島素打開細胞的「門」，把血糖送進去利用或儲存。本質是合成型荷爾蒙——身體進入「建造與儲存」模式。
  • 3
    燃脂反應在胰島素分泌時暫停
    胰島素會抑制脂肪分解酶（HSL）的活性，讓脂肪細胞無法釋出脂肪酸。胰島素高 = 囤積優先、燃脂暫停。

⚠️ 重要提醒：不是「胰島素越低越好」。
餐後升高、空腹回落，是身體正常的代謝節律。重點是節律是否正常、身體對胰島素的反應是否靈敏——不是讓它一直保持低點。

第二頁 · 阻抗的意思

「阻抗」是什麼意思？
為什麼跟變胖有關？

阻抗代表細胞的「門」變不靈敏了，身體需要分泌更多胰島素才能達到同樣效果。

🔴 核心觀念：
胰島素長時間維持在高水位 → 身體持續處於「囤積優先、燃脂暫停」的狀態。
脂肪不容易動用、多餘熱量更快進脂肪細胞，這才是「容易胖」的真正機制。

🫃

主因：內臟脂肪

內臟脂肪釋放的發炎物質（TNF-α、IL-6）直接干擾胰島素受體訊號。腰圍比體脂率更能反映風險。

😴

常被低估：睡眠壓力

皮質醇長期偏高 → 直接促進內臟脂肪累積，並造成功能性胰島素阻抗。影響力遠大於偶爾吃甜食。

🪑

靜態生活

骨骼肌是最大的葡萄糖利用器官。缺乏活動 → 肌肉不參與代謝 → 胰島素需要做更多工作。

🌿 關鍵觀念：它是連續的、可逆的代謝狀態。
不是你基因決定好的命運。成因和嚴重程度，幾乎完全取決於整體的生活習慣。

第四頁 · 不同族群

你的條件，
決定阻抗的影響力有多大

同樣叫「胰島素阻抗」，對不同人的實際意義差距非常大。點選符合你的狀況。

胰島素阻抗的影響力

  • 內臟脂肪高，阻抗程度可能較明顯
  • 熱量波動大時，脂肪合成效率更高
  • 蛋白質不足時優先分解肌肉
  • 每次大起大落都在強化惡性循環

優先策略

  • 穩定攝取比製造大赤字更重要
  • 蛋白質優先：每餐至少 20–25g
  • 步數 ≥ 7,000，日常活動優先
  • 改善睡眠品質，降低皮質醇
策略不是迴避碳水，而是建立規律節奏——讓身體重新感受到安全感。

胰島素阻抗的影響力

  • 運動習慣大幅改善胰島素敏感性
  • 肌肉是天然的葡萄糖緩衝器
  • 此時阻抗影響力已大幅降低
  • 腰圍正常者代謝風險通常不高

真正卡關的常見原因

  • 熱量被低估（油脂、醬料、飲料）
  • 蛋白質不足（目標 1.6–2g/kg）
  • 整體 NEAT 活動量被高估
  • 睡眠不足或慢性壓力（皮質醇）
卡關通常跟胰島素阻抗關係不大。先查飲食記錄完整度、蛋白質量、睡眠——三件事影響力加起來遠大於荷爾蒙。

胰島素阻抗的影響力

  • 在這個體脂範圍，影響力極小
  • 規律運動已持續改善敏感性
  • 偶爾吃甜食的干擾非常有限
  • 擔心阻抗通常是資訊過載的結果

真正值得關注的

  • 體脂越低，身體防禦越強（瘦素下降）
  • 蛋白質是否足夠維持肌肉量
  • 是否有真實的熱量缺口
  • 訓練是否有漸進性刺激
把精力放在訓練品質和蛋白質攝取上，效果遠大於繼續擔心胰島素阻抗。

實際發生的機制

  • 長期低熱量 → 瘦素下降 → 代謝率降低
  • 肌肉流失 → 基礎代謝持續下降
  • 皮質醇偏高 → 功能性胰島素阻抗
  • 溜溜球循環強化「易胖體質感」

優先處理

  • 不是「吃更少」，是「吃得更對」
  • 逐步提升蛋白質攝取，保留肌肉
  • 導入阻力訓練，重建代謝底盤
  • 修復飲食節律（規律餐次優先）
打破循環的起點不是更嚴格，而是更穩定。這個族群的阻抗，很大程度是長期節食和壓力造成的功能性狀態。

備忘筆記｜原理速查

胰島素阻抗：原理備忘

講解時一時想不起某個機制，切過來看。

觸發胰島素分泌的不只碳水

什麼會讓胰島素升高？

觸發來源機制強度
碳水化合物血糖直接升高 → 胰臟 β 細胞分泌胰島素最強
蛋白質BCAA、亮胺酸（Leucine）刺激 β 細胞中等
脂肪（單獨）幾乎不刺激極低
脂肪＋碳水同吃延遲胃排空 → 血糖波動拉長 → 胰島素延長分泌中高
頭期相分泌看到/聞到食物 → 大腦預先觸發少量分泌低
某些甜味劑腸道激素間接路徑，研究仍有爭議不確定

教學重點：不要說「只要吃碳水就會胰島素高」——蛋白質也會觸發，脂肪＋碳水同吃會放大。正確說法是「進食都會觸發，碳水最直接，但整體飲食模式比單一食物更重要。」

合成型荷爾蒙機制

胰島素同時在做什麼？

① 促進葡萄糖進入細胞（肌肉、脂肪、肝臟）→ 利用或儲存為肝醣

② 促進蛋白質合成（肌肉修復、增長）

③ 促進脂肪合成（多餘熱量轉為三酸甘油酯儲存）

④ 抑制脂肪分解（抑制 HSL：hormone-sensitive lipase）

常見誤解要糾正：胰島素不是「越低越好」。第一型糖尿病完全缺乏胰島素反而會危及生命。重點是節律正常性與細胞反應靈敏性，不是讓它一直保持低點。

內臟脂肪如何造成阻抗

細胞的「門」為什麼變不靈敏？

內臟脂肪（≠ 皮下脂肪）是代謝活性高的組織，釋放：

• TNF-α：干擾胰島素受體底物（IRS-1）磷酸化，受體訊號傳導出問題

• IL-6：系統性發炎，降低肌肉和肝臟的胰島素敏感性

• 游離脂肪酸（FFA）：大量進入肝臟和肌肉，干擾葡萄糖代謝路徑

快速記憶：腰圍偏大 → 內臟脂肪多 → 發炎訊號高 → 胰島素受體鈍化 → 需要分泌更多 → 血糖反應變慢 → 脂肪更容易囤積

運動的兩條路

肌肉為什麼是天然解藥？

機制說明效果持續
GLUT4 上調（急性）肌肉收縮本身不依賴胰島素，直接把 GLUT4 轉運體送到細胞膜，讓葡萄糖進入運動後 24–48 小時
肌肉量提升（慢性）肌肉量越多 → 全身葡萄糖緩衝容量越大 → 胰島素需要做的工作越少長期累積效果

教學語言建議：「你每次做重訓或有氧，肌肉就在幫你清血糖——而且這個效果在運動後還會持續將近兩天。所以有運動習慣的人，身體本來就在持續對抗胰島素阻抗的形成。」

溜溜球惡性循環

熱量大起大落為什麼特別不利？

高熱量的天：血糖大幅震盪 → 胰島素大量分泌 → 脂肪合成訊號強 → 多餘熱量幾乎直接進脂肪細胞（因為已有阻抗，這個過程比一般人更快）

低熱量的天：缺乏能量＋蛋白質不足 → 身體優先分解肌肉（比脂肪更容易動用）→ 肌肉量減少 → 基礎代謝下降 → 下次吃多一點更容易胖

結論：每一次的大起大落，都在強化「胰島素阻抗 + 肌肉流失 + 代謝變差」的三角惡性循環。這是溜溜球體質越來越難瘦的原因——不是意志力不夠，是代謝環境被訓練成更容易囤積。

皮質醇的角色

睡眠與壓力的影響機制

慢性睡眠不足和高壓 → 皮質醇長期偏高，皮質醇會：

① 直接促進內臟脂肪囤積（尤其腹部）

② 抑制脂肪氧化（身體更傾向燃燒碳水和肌肉）

③ 造成功能性胰島素阻抗（即使體脂不高也會發生）

④ 升高血糖（刺激肝臟釋放葡萄糖 → 胰島素跟著升高）

重點：「明明吃得不多但體重不動」的人，睡眠和壓力是最常被忽略的關鍵變數。飲食記錄看起來沒問題，但皮質醇在阻礙代謝。

對容易焦慮學生的語言框架

怎麼跟對號入座的人說？

「胰島素阻抗這件事，在你現在的條件下影響力其實很小。你有運動習慣，肌肉本身每天都在幫你維持胰島素敏感性。真正值得我們一起看的是：你的熱量缺口夠不夠、蛋白質有沒有到位、睡眠和壓力狀態怎麼樣——這三件事對你體脂的影響力，加起來是胰島素阻抗的好幾倍。」

不要說：「你的胰島素阻抗問題需要特別注意」（體脂正常、有運動的人聽到會不必要地焦慮）

要說：「你現在需要的不是擔心荷爾蒙，而是把基本功做扎實——蛋白質、睡眠、熱量缺口。」
