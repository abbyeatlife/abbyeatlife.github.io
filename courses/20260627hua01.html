🔬 為什麼蛋白質跟情緒有關？
  • 血清素（Serotonin）是影響情緒穩定的重要神經傳導物質

色胺酸與血清素的完整機制 給老師自己理解用

色胺酸是什麼？

色胺酸（Tryptophan）是一種必需胺基酸——人體無法自行合成，只能從食物中攝取。含量較多的食物包括：蛋、豆腐、豆漿、雞胸肉、鮭魚、堅果、香蕉。

從食物到血清素的路徑

食物中的蛋白質
→
消化分解出色胺酸
→
進入血液
→
競爭穿越血腦屏障
→
進入大腦
→
色胺酸 → 5-HTP
→
5-HTP → 血清素

為什麼「競爭穿越血腦屏障」這步很關鍵？

色胺酸要進入大腦，需要通過一個載體蛋白，但這個載體同時也要運輸其他大型中性胺基酸（LNAA，如苯丙胺酸、白胺酸、纈胺酸等）。這些胺基酸和色胺酸共用同一條通道，互相競爭。

這就解釋了一個很反直覺的現象：

  • 吃純蛋白質 → 所有胺基酸一起升高 → 色胺酸競爭輸 → 進腦量反而沒增加多少
  • 吃蛋白質＋少量碳水 → 胰島素分泌 → 把其他競爭者（LNAA）送進肌肉 → 色胺酸「獨走」進大腦 → 血清素合成增加

這也是為什麼人情緒低落時會本能地想吃澱粉——大腦在用一個笨方法幫自己補血清素。

跟學生溝通時：不需要講競爭機制這麼細，說「蛋白質是血清素的原料，但要配合一點澱粉才吸收得好」就夠了，而且剛好對應她目前的飲食問題（純澱粉早餐 = 沒有原料；純蛋白質也不是最佳 = 需要搭配）。
血清素與情緒：要小心的說法 科學上有一些細節

「血清素低 = 憂鬱」這個說法是過度簡化的

這個「血清素假說（Serotonin Hypothesis）」從 1960 年代開始流行，也是 SSRI 藥物設計的理論基礎。但近年（尤其 2022 年後）有多篇 meta-analysis 指出，直接把「血清素濃度低」等同於「憂鬱症」的說法，在科學上支持度其實不強。

  • 憂鬱症是多因素的，血清素只是其中一條路
  • 更精確的說法：血清素系統影響的是情緒調節的靈活度和韌性，而非直接等於快樂或不快樂
  • SSRI 藥物對部分人有效，但機制可能不只是「補血清素」這麼單純
對 Hua 的應用：不要說「妳血清素太低所以憂鬱」（這既不精確也不是妳該說的）。可以說的是：「飲食裡蛋白質足夠，有助於讓神經系統有足夠的材料運作，情緒的穩定性也會比較好。」這是有根據且在妳範疇內的說法。
腸道、過敏、血清素：三者的連結 Hua 特別相關

腸道是血清素的主要製造地

  • 人體約 90% 的血清素在腸道製造（由腸嗜鉻細胞 Enterochromaffin cells 產生）
  • 但腸道的血清素無法穿越血腦屏障，所以它主要負責腸道蠕動和腸道神經的局部調控
  • 腸道影響大腦情緒的主要路徑是透過迷走神經（Vagus nerve），也就是「腸腦軸（Gut-Brain Axis）」

→

→

→

Hua 的過敏跟這條線有什麼關係？

  • 皮膚過敏 = 免疫系統過度活化 = 身體處於慢性發炎狀態
  • 慢性發炎本身就會加重腸道黏膜的損傷（腸道菌相和皮膚免疫是雙向影響的）
  • 這也是為什麼「過敏體質的人」比較容易有腸胃敏感、腸躁等問題
  • 飲食中的精緻糖和高度加工食品 → 加劇腸道菌相失衡 → 皮膚和情緒雙重惡化
給 Hua 的實用結論：降低精緻糖和加工食品，對她的皮膚、腸道、和情緒穩定性同時有幫助——這是同一件事，不是三件事。這個論點你可以在課程中用來串連她提出的三個問題，讓她覺得「原來這些都是相關的」，不是一堆互相獨立的問題。
黃體期血糖變差——是所有女生都這樣嗎？ 答案：是，但程度差很多

生理機制

  • 黃體期（排卵後到月經前）黃體素（Progesterone）濃度升高
  • 黃體素本身有輕度抗胰島素作用——讓細胞對胰島素的敏感性下降
  • 身體需要分泌更多胰島素才能維持同樣的血糖穩定
  • 月經前黃體素驟降 → 這個落差期的血糖調節最不穩定 → 同時血清素和多巴胺也在這個時期偏低

為什麼有些人特別嚴重？

  • 平時飲食精緻碳水比例高 → 血糖本來就不穩，黃體期更放大
  • 慢性壓力大 → 皮質醇升高也會干擾血糖調節，疊加效應
  • 睡眠差 → 同上，睡眠不足本身就讓血糖更難控制
  • 基礎蛋白質攝取不足 → 血糖起伏更大

Hua 目前這四個因素可能都有，所以她的症狀比一般人明顯是合理的——不是她特別脆弱，是她的處境讓這個生理機制被放大了。

跟 Hua 說這段的方式：「這個不是只有妳這樣，幾乎所有女生月經前都會有程度不一的血糖不穩定，只是有人輕到沒感覺。妳之所以特別明顯，是因為妳同時還有其他因素在加重它——這些都是可以慢慢改善的。」先讓她覺得自己不是異常，再談策略。
高糖 → 胰島素 → 發炎（IL-6、TNF-α）的機制 皮膚過敏的飲食連結

問題不是胰島素本身，而是血糖波動太劇烈

胰島素的本職是把血糖送進細胞，這是正常且必要的。問題在於血糖飆升太快太高時，觸發的一連串下游反應。

路徑一：脂肪細胞的發炎訊號

胰島素大量分泌
→
促進脂肪合成
脂肪細胞變大
→
分泌 IL-6、TNF-α
→
活化更多免疫細胞
→
發炎循環

IL-6（介白素-6）和 TNF-α（腫瘤壞死因子）不只是發炎的標記，它們本身就是發炎的推手，會持續活化免疫系統。

路徑二：氧化壓力

細胞處理大量葡萄糖
→
粒線體過載
→
產生大量自由基（ROS）
→
活化 NF-κB（發炎開關）
→
更多 IL-6、TNF-α

路徑三：腸道菌相擾動

高糖飲食
→
腸道菌相失衡
→
黏膜屏障下降
→
LPS（細菌碎片）滲漏進血液
→
強烈活化免疫系統

LPS（脂多醣）是革蘭氏陰性菌的細胞壁碎片，是目前已知最強的發炎觸發物之一。

這些發炎物質對皮膚做了什麼

  • 活化肥大細胞（Mast cells）→ 釋放組織胺 → 過敏反應加重
  • 破壞皮膚屏障蛋白（Filaggrin）→ 皮膚通透性上升 → 外來刺激更容易滲入
  • 促進角質細胞過度增生 → 部分人表現為痘痘或濕疹
對 Hua 的意義：她本來就有免疫系統過度活化的底子，飲食上持續給高糖刺激，等於一直在按發炎的油門，讓皮膚屏障沒有機會修復。降低精緻糖不是「保養」，是在減少不必要的免疫系統負擔。

跟她說的話：「糖吃多了，身體的免疫系統會一直處於警備狀態，妳的過敏就更容易被觸發，而且這個反應是有機制的，不是迷思。」
糖化（Glycation）與膠原蛋白、皮膚老化 AGEs 的機制

什麼是糖化？

當血液裡有過多的葡萄糖或果糖，這些糖分子會非酶性地附著在蛋白質上，產生 AGEs（Advanced Glycation End-products，糖化終產物）。

這個反應不需要酵素，只要糖和蛋白質接觸、時間夠長就會發生。你在廚房看到食物加熱後變褐色（梅納反應），本質上是同一件事——只是在體內是在體溫下慢慢、持續地進行。

糖化對膠原蛋白做了什麼

血液中多餘的糖
→
附著在膠原蛋白上
→
纖維之間產生交聯
（Cross-linking）
→
膠原蛋白變硬
失去彈性
→
皺紋、鬆弛
失去光澤

正常膠原蛋白纖維柔軟有彈性。糖化後纖維互相黏連，就像橡皮筋被黏在一起就沒辦法伸縮。更麻煩的是，受損的膠原蛋白很難被身體替換掉——膠原蛋白的代謝週期本來就很慢。

AGEs 還會再次觸發發炎

AGEs 在體內累積
→
與細胞上的 RAGE 受體結合
→
活化 NF-κB
→
又產生 IL-6、TNF-α
→
回到發炎循環

也就是說，糖化產物本身又成為新的發炎來源——兩條路徑最終匯流到同一個結果。

⚠️ 果糖特別需要注意

果糖的糖化速度比葡萄糖快 7–10 倍，而果糖大量藏在：含糖飲料、市售果汁、加工食品裡的高果糖漿（HFCS）。「不甜的加工食品」裡可能藏了比你想像更多的果糖。

兩條路徑總整理

高糖飲食
├─→ 血糖快速飆升
│     ├─→ 胰島素大量分泌 → 脂肪細胞活化 → IL-6、TNF-α → 全身發炎
│     ├─→ 氧化壓力（ROS）→ NF-κB → 更多發炎
│     └─→ 腸道菌相失衡 → LPS 滲漏 → 免疫活化
└─→ 血液中多餘的糖
         └─→ 糖化（Glycation）→ AGEs
                 ├─→ 膠原蛋白交聯 → 失彈性、皺紋
                 └─→ RAGE 活化 → 回到發炎路徑
跟 Hua 說的話：「糖對皮膚有兩個壞處：一個是讓免疫系統一直處於警備狀態，過敏就更容易被觸發；另一個是糖會直接黏在皮膚的膠原蛋白上，讓它變硬、失去彈性。這兩件事同時在發生，所以吃糖對皮膚的影響不是迷思，是有機制的。」
